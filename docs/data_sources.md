# Data Sources

## Pretraining PGN Source

A portion of the multi-player pretraining PGNs in this project comes from:

- The Week in Chess (TWIC): https://theweekinchess.com/twic

Use of TWIC data should follow their terms and attribution expectations.

## Fine-tuning Target-Player Game Source

Target-player games for fine-tuning are collected from:

- Chess.com Games archive/search pages: https://www.chess.com/games

This reference documents the provenance of target-player game collection used for adaptation datasets.
